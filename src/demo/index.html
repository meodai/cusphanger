Cusp­Hanger

A RampenSau-style API for creating sequential and diverging palettes from just a few intuitive parameters using a quadratic Bézier curve through a hue's gamut triangle, sampled at perceptual lightness steps.

A faithful OKLCH implementation of Wijffelaars, Vliegen, van Wijk & van der Linden, “Generating Color Palettes using Intuitive Parameters” (Computer Graphics Forum 28:3, EuroVis 2009). Also inspired by Matt DesLauriers’s OKLCH take on the same paper. Gamut boundaries (the cusp / shell) come from nutelch.
